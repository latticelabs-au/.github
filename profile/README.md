## 🧭 What we build

**[Lattice Look](https://latticelook.au)** is our flagship platform: multi-tenant compliance and learning management built for Australian councils, government agencies and enterprise. Everything runs on an Australian-hosted, high-availability stack with PostgreSQL row-level security tenant isolation and a tamper-evident audit trail.
