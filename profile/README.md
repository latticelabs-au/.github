## What we build

**[Lattice Look](https://latticelook.au)** is our flagship platform: multi-tenant compliance and learning management built for Australian councils, government agencies and enterprise. It covers training compliance and LMS delivery, WHS (inductions, inspections, hazards, PPE, contractor management), ISO 9001/45001 document control, GRC registers and public events, all on an Australian-hosted, high-availability stack with PostgreSQL row-level security tenant isolation and a tamper-evident audit trail.
